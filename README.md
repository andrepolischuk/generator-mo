# generator-mo [![Build Status][travis-image]][travis-url]

  > Yeoman generator for node and browser modules

  With CLI and ES2015 syntax support

```sh
yo mo
```

## Install

```sh
npm install --global generator-mo
```

## Usage

```sh
yo mo
```
